# Student Result Analyzer

A GUI-based Python application for analyzing student academic performance.

The application allows users to enter student marks, calculate results, generate rank lists, and store data using CSV files. It also includes a dark mode option for improved user experience.

## Features
- Add student details and marks
- Automatic calculation of total, average, grade, and result
- Rank list generation based on total marks
- View all student records
- CSV file storage for data persistence
- Dark mode toggle
- User-friendly graphical interface

## Technologies Used
- Python
- Tkinter (GUI)
- CSV file handling
- Git & GitHub
